remake works as a command-line tool, and as a Python package.

remake setup-examples
cd remake-examples
remake run ex1
cat README.md

Alternatively, any remake file can be run using Python (e.g. using IPython):

run ex1.py
ex1.finalize()
ex1.run_all()

remake
